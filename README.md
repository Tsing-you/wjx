# 问卷星自动化油猴脚本

## 项目概述

这是一个基于 Tampermonkey 的用户脚本，用于自动化填写问卷星平台上的问卷。该项目在原作者 Zemelee 的基础上进行了改进，特别增加了对阿里云验证码（点击）的应对机制。无视多页问卷限制。可通过多标签页多开。

### 项目特点

- **自动化填写**：支持多种题型的自动填写
- **智能反检测**：隐藏 webdriver 标识，模拟真实浏览器指纹
- **验证码处理**：集成阿里云验证码自动点击功能
- **数据统计**：自动记录已完成问卷数量

## 功能特性

### 支持的题型
- ✅ 单选题 (`single`)
- ✅ 多选题 (`multiple`) 
- ✅ 矩阵题 (`matrix`)
- ✅ 量表题 (`scale`)
- ✅ 滑块题 (`slide`)
- ✅ 填空题 (`vacant`)
- ✅ 排序题 (`reorder`)
- ✅ 地理位置题 (`setLocation`)

### 反检测机制
- 清除本地存储和会话数据
- 隐藏 webdriver 检测标记
- 随机化浏览器指纹信息
- 屏蔽脚本检测

### 验证码处理
- 模拟真人操作轨迹
- 长按验证按钮
- 自适应等待时间

## 安装要求

### 浏览器扩展
- [Tampermonkey](https://www.tampermonkey.net/) 或其他用户脚本管理器

### 依赖库
- [localforage](https://cdnjs.cloudflare.com/ajax/libs/localforage/1.7.3/localforage.min.js) - 用于本地数据持久化

## 安装步骤

1. 安装 Tampermonkey 浏览器扩展
2. 将脚本代码复制到新的用户脚本中
3. 修改 `@match` 规则以匹配您的问卷网址
4. 替换脚本中的问卷链接：

```javascript
var url = "https://v.wjx.cn/vm/xxxxxx.aspx"; // 替换为实际问卷链接
```

5. 配置题目设置（根据实际问卷调整）

## 使用说明

### 题目配置规则

#### 单选题
```javascript
single(题号, [选项权重比例])
// 例：single(1, [18, 42, 15, 13, 8, 4])
```

#### 多选题
```javascript
multiple(题号, [选项选择比例])
// 例：multiple(2, [78, 18, 42, 12, 35, 0])
```

#### 滑块题
```javascript
slide(题号, 最小值, 最大值)
// 例：slide(7, 2, 5)
```

#### 填空题
```javascript
vacant(题号, [文本选项], [选择比例])
// 例：vacant(26, ["无/没有/不了解", "具体建议"], [85, 15])
```

#### 矩阵题
```javascript
matrix(题号, {子题选项比例})
// 例：matrix(4, {"1": [1, 0, 0, 0, 0], ...})
```

### 重要提示
- 题目数量和比例数量必须一致
- 所有符号需使用英文输入法
- 完成配置后按 `Ctrl+S` 保存

## 配置技巧

### 快速获取题目信息
1. 访问作者推荐的解析网站：sugarblack.top
2. 输入问卷链接并点击解析
3. 复制比例按钮获取数据

### 脚本运行
- 启用插件后，在新标签页中打开问卷链接

## 技术实现

### 主要功能模块
- **存储管理**：使用 localforage 进行数据持久化
- **DOM 操作**：通过 XPath 和 CSS 选择器定位元素
- **事件模拟**：模拟真实的鼠标和键盘操作
- **异步处理**：使用 async/await 处理异步操作

### 验证码处理流程
1. 等待验证码元素加载
2. 模拟鼠标在验证框周围随机移动
3. 精准定位到验证按钮中心
4. 执行长按操作模拟真人行为
5. 验证处理结果
